# Week 04 — Binary Search, Trees, and Time-Series Analysis

## Overview

Covers binary search algorithms, binary tree structures (BST, traversals), and applied signal-processing techniques including time-series plotting and moving-average filtering.

## Contents

| Path | Description |
|------|-------------|
| `binary-search.ipynb` | Binary search on sorted arrays — iterative and recursive implementations |
| `binary-trees.ipynb` | Binary search trees — insertion, lookup, in/pre/post-order traversal |
| `time-series-analysis/plot-signals.py` | Signal generation and visualization with Matplotlib |
| `time-series-analysis/moving-average-filter.py` | Moving-average smoothing filter applied to noisy signals |
| `time-series-analysis/plot-functions-forloop.py` | Plotting mathematical functions using iterative loops |
| `time-series-analysis/sum-and-trianglular-numbers.py` | Summation sequences and triangular number computation |

## Key Concepts

- Binary search — O(log n) divide-and-conquer lookup
- Binary search trees — ordered insertion, balanced vs unbalanced trees
- Tree traversals — in-order, pre-order, post-order
- Time-series analysis — signal generation, noise, moving-average filters

## How to Run

```bash
# Notebooks
jupyter notebook binary-search.ipynb
jupyter notebook binary-trees.ipynb

# Time-series scripts (requires NumPy, Matplotlib)
python time-series-analysis/plot-signals.py
python time-series-analysis/moving-average-filter.py
```
