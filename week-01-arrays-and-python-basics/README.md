# Week 01 — Arrays and Python Basics

## Overview

Foundational Python programming and introductory data structures. Covers binary number representation, static arrays, input handling, unit conversion programs, and general Python syntax review.

## Contents

| Path | Description |
|------|-------------|
| `getting-started/` | Starter programs — temperature/distance converters, number checker, and their unit tests |
| `binary.ipynb` | Binary number representation and bitwise operations |
| `static-arrays.ipynb` | Introduction to static arrays and array-based algorithms |
| `python-course.ipynb` | General Python review — variables, control flow, functions, I/O |
| `human.py` | Standalone script demonstrating class-based modelling |

## Key Concepts

- Python fundamentals (variables, conditionals, loops, functions)
- Binary representation and base conversion
- Static arrays — indexing, traversal, and basic operations
- Unit testing with `pytest`

## How to Run

```bash
# Notebooks
jupyter notebook binary.ipynb

# Scripts
python getting-started/temperature_converter.py

# Tests
pytest getting-started/
```
