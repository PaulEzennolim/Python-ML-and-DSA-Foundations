# Week 03 — Linked Lists and Loop-Based Programs

## Overview

Implements singly and doubly linked lists from scratch and applies iterative programming patterns through standalone loop-based programs.

## Contents

| Path | Description |
|------|-------------|
| `linked-lists.ipynb` | Singly and doubly linked list — node creation, insertion, deletion, traversal, reversal |
| `using-loops/factorial.py` | Iterative factorial computation |
| `using-loops/guessing_game.py` | Number-guessing game using a while loop |
| `using-loops/newtons-methods-roots.py` | Newton's method for approximating square roots |

## Key Concepts

- Singly linked lists — head pointer, node chaining, O(1) insert at head
- Doubly linked lists — forward/backward traversal, bidirectional pointers
- Iterative algorithms — factorial, numerical methods
- Newton–Raphson root-finding method

## How to Run

```bash
# Notebook
jupyter notebook linked-lists.ipynb

# Scripts
python using-loops/factorial.py
python using-loops/guessing_game.py
python using-loops/newtons-methods-roots.py
```
